# Codveda Data Analytics Internship Portfolio

This repository contains my projects completed during the Data Analytics Internship at Codveda Technology. The portfolio is structured into three levels of complexity, ranging from basic data processing to advanced predictive modeling.

## 🛠 Tools & Technologies
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, TextBlob, Statsmodels
- **Environment:** Google Colab / Jupyter Notebook

## 📂 Project Structure

### Level 1: Basic
- **Task:** Data Cleaning & Exploratory Data Analysis (EDA).
- **Project:** Analyzing Customer Churn to identify early warning signs of customer attrition.

### Level 2: Intermediate
- **Task 1:** Regression Analysis for House Price Prediction.
- **Task 2:** Time Series Analysis for Stock Price Trends using Forward Fill (.ffill) and Moving Averages.

### Level 3: Advanced
- **Task 1:** Predictive Modeling using Random Forest Classifier for Churn prediction (95.5% Accuracy).
- **Task 3:** NLP Sentiment Analysis using TextBlob to classify public perception into Positive, Neutral, and Negative.

## 📈 Key Insights
- Successfully built a predictive model with high precision to mitigate business risks.
- Implemented advanced time-series techniques to isolate market trends from noise.
- Extracted meaningful sentiment patterns from raw textual data.
